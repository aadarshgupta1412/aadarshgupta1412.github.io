# Aadarsh Gupta - Portfolio

Personal portfolio built with Next.js 14, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with OKLCH color system
- **Animations**: Framer Motion
- **Deployment**: GitHub Pages (static export)

## Quick Start

```bash
pnpm install
pnpm dev
```

## Structure

```
app/           # Pages (home, about, projects, contact)
components/    # UI components (Accordion, Tabs, Navigation, etc.)
lib/           # Data and utilities
public/        # Static assets
```

## Features

- Dark/light theme with system preference detection
- Responsive design (vertical nav on desktop, hamburger on mobile)
- Auto-hiding scrollbar
- Accordion & Tabs components for organized content
- Accessible with keyboard navigation and reduced motion support

## Deploy

Push to `main` triggers GitHub Actions deployment to GitHub Pages.
